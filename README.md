# Shadow of the Demon King

> A LOTR-inspired horror indie game with atmospheric exploration, challenging combat, and an epic demon king final boss.

## 🎮 Overview

This repository contains the complete game design document and project structure for **Shadow of the Demon King** — a dark fantasy horror game where you play as The Last Warden, carrying the Flame of Hope through six cursed realms to defeat the Demon King.

## 🌟 Key Features

- **6 Unique Levels:** From forsaken villages to volcanic wastelands
- **Epic Boss Fights:** Each level culminates in a challenging boss encounter
- **The Demon King:** 4-phase final boss inspired by epic fantasy battles
- **Atmosphere:** LOTR-inspired dark fantasy meets survival horror
- **Core Mechanic:** The Flame of Hope — a risk/reward lantern system

## 🗺️ Level Structure

1. **The Forsaken Village** — Quiet horror introduction
2. **The Blighted Woods** — Twisted forest maze
3. **The Shattered Catacombs** — Claustrophobic underground
4. **The Ruined City of Khar-Dûm** — Gothic architecture horror
5. **The Ashen Wasteland** — Volcanic Mordor-inspired plains
6. **The Obsidian Throne** — Final fortress and demon king battle
